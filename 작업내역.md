# 클락(앙헬레스) 여행 플래너 — 작업 내역

가이드형 페이지를 **개인 전용 여행 플래너**로 개편하고, 골프 비용 계산기·환율·지출 관리를 통합한 작업 기록입니다.

## 구성

- `index.html` / `trip.html` — 동일한 단일 페이지 앱 (두 파일 동기화 유지)
- `app.js` — 모든 로직 (바닐라 JS, 빌드 불필요)
- `styles.css` — 스타일
- `clark_golf_calculator.xlsx`, `클락(앙헬레스)가이드.xlsx` — 원본 데이터(참고용)

모든 데이터는 브라우저 `localStorage`에 저장됩니다. 서버·계정 없이 동작합니다.

## 탭 구성

| 탭 | 내용 |
|----|------|
| **일정** | 하루씩 보는 타임라인. 일차 탭으로 전환. 항목 추가/수정/삭제, 항목별 예산(₱) 입력 |
| **지출** | 환율 설정 + 일차별·전체 지출을 페소·원·달러로 표기 |
| **골프** | 골프장·요일·인원 선택 → 1인당/전체 비용 자동 계산 |
| **메모** | 자유 텍스트 + 이미지 붙여넣기(Ctrl+V) |

## 주요 변경 사항

### 1. 가이드 → 전용 여행 플래너로 개편
- 제목 "여행 가이드" → "여행 플래너", 브랜드 "My Clark Trip"
- 메인(일정) 화면에서 헤더의 예상 지출 금액, 일차별 합계 등 **상세 지출 표기 제거**
- "전체(모두 보기)" 제거 → **하루씩만** 표시

### 2. 여행 기간 입력 (몇 박 몇 일)
- 상단에 `박` / `일` 입력칸 추가
- 입력한 **일수만큼 1일차~N일차 탭 자동 생성**
- 한쪽을 바꾸면 다른 쪽 자동 보정(일수 = 박수 + 1), 직접 수정도 가능
- 기본값 3박 4일

### 3. 골프 비용 계산기 통합 (`clark_golf_calculator.xlsx` 이전)
- 13개 골프장 요금표(그린피 주중/주말·캐디·인원별 카트·보험·컨슈머블) 내장
- 인원수(2/3/4)·주중/주말 선택 → 라운드별 1인 비용 자동 분해 + N라운드 합계/평균/전체 합계
- 엑셀 계산 결과와 일치 검증 완료 (예: 3R 합계 ₱11,850 / 평균 ₱3,950 / 4명 전체 ₱47,400)
- 골프장 요금표 펼쳐보기 포함

### 4. 환율 (원·달러·페소 모두 표기)
- 지출·골프의 모든 금액을 **₱ / ₩ / $** 로 함께 표기
- `₱→₩`, `₱→$` 환율 **직접 입력**(저장됨)
- **환율 자동 갱신** 버튼: open.er-api.com에서 실시간 환율을 받아옴
  - ⚠️ 인터넷 연결이 필요. 오프라인(여행 중)이거나 실패 시 수동 입력값을 사용

### 5. 장소 페이지 삭제
- 구글맵 장소 목록 탭/화면 및 관련 기능 전부 제거

### 6. 메모 내용 초기화
- 기본 시드(준비물 체크리스트 등) 제거, 빈 메모로 시작 (저장 키 v2)

## 데이터 관리
- **PDF로 저장 / 공유**: 모든 탭을 한 문서로 펼쳐 인쇄
- **백업 내보내기 / 가져오기**: 일정·지출·환율·골프·여행기간을 JSON으로
- **기본값으로 초기화**: 저장 데이터 삭제 후 기본 일정 복원
