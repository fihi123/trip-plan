# 클락(앙헬레스) 여행 플래너 — 작업 내역

가이드형 페이지를 **개인 전용 여행 플래너**로 개편하고, 골프 비용 계산기·환율·지출 관리를 통합한 작업 기록입니다.

## 구성

- `index.html` / `trip.html` — 동일한 단일 페이지 앱 (두 파일 동기화 유지)
- `app.js` — 모든 로직 (바닐라 JS, 빌드 불필요)
- `styles.css` — 스타일
- `clark_golf_calculator.xlsx`, `클락(앙헬레스)가이드.xlsx` — 원본 데이터(참고용)

모든 데이터는 브라우저 `localStorage`에 저장됩니다. 서버·계정 없이 동작합니다.

## 탭 구성

| 탭 | 내용 |
|----|------|
| **일정** | 하루씩 보는 타임라인. 일차 탭으로 전환. 항목 추가/수정/삭제, 예산(₱)·지출 분류 선택. 배정된 골프 라운드 자동 표시. 여정 PDF 불러오기 |
| **지출** | 환율 설정 + 카테고리별·일차별·항목별 지출(일정+골프 합산)을 페소·원·달러로 표기 |
| **골프** | 골프장·요일·인원·일정 일차 선택 → 1인당/전체 비용 자동 계산, 요금표 직접 수정 |
| **메모** | 자유 텍스트 + 이미지 붙여넣기(Ctrl+V) |

## 주요 변경 사항

### 1. 가이드 → 전용 여행 플래너로 개편
- 제목 "여행 가이드" → "여행 플래너", 브랜드 "My Clark Trip"
- 메인(일정) 화면에서 헤더의 예상 지출 금액, 일차별 합계 등 **상세 지출 표기 제거**
- "전체(모두 보기)" 제거 → **하루씩만** 표시

### 2. 여행 기간 입력 (몇 박 몇 일)
- 상단에 `박` / `일` 입력칸 추가
- 입력한 **일수만큼 1일차~N일차 탭 자동 생성**
- 한쪽을 바꾸면 다른 쪽 자동 보정(일수 = 박수 + 1), 직접 수정도 가능
- 기본값 3박 4일

### 3. 골프 비용 계산기 통합 (`clark_golf_calculator.xlsx` 이전)
- 13개 골프장 요금표(그린피 주중/주말·캐디·인원별 카트·보험·컨슈머블) 내장
- 인원수(2/3/4)·주중/주말 선택 → 라운드별 1인 비용 자동 분해 + N라운드 합계/평균/전체 합계
- 엑셀 계산 결과와 일치 검증 완료 (예: 3R 합계 ₱11,850 / 평균 ₱3,950 / 4명 전체 ₱47,400)
- 골프장 요금표 펼쳐보기 포함
- **요금표 직접 수정 가능**: 골프장명·그린피(주중/주말)·캐디·카트·보험+컨슈 값을 표에서 바로 편집하면 저장되어 라운드 계산에 즉시 반영. 카트는 현재 선택 인원수 기준으로 편집. "기본값으로 되돌리기" 버튼 제공

### 4. 환율 (원·달러·페소 모두 표기)
- 지출·골프의 모든 금액을 **₱ / ₩ / $** 로 함께 표기
- `₱→₩`, `₱→$` 환율 **직접 입력**(저장됨)
- **환율 자동 갱신** 버튼: open.er-api.com에서 실시간 환율을 받아옴
  - ⚠️ 인터넷 연결이 필요. 오프라인(여행 중)이거나 실패 시 수동 입력값을 사용

### 5. 장소 페이지 삭제
- 구글맵 장소 목록 탭/화면 및 관련 기능 전부 제거

### 6. 메모 내용 초기화
- 기본 시드(준비물 체크리스트 등) 제거, 빈 메모로 시작 (저장 키 v2)

### 7. 여정안내서(PDF) 자동 인식
- 일정 탭의 **"여정 PDF 불러오기"**로 전자항공권(이티켓) PDF 업로드
- **출국일·귀국일·일수(박/일) 자동 설정** + 출국일 기준 각 일차에 실제 날짜(요일) 표시
- **비행편을 타임라인에 자동 추가**: 편명·구간·출발/도착 시각을 해당 일차 일정으로 (예: `✈ OZ707 출국 SEOUL INCHEON→CLARK` 21:55→00:50)
- **항공료 자동 인식**: e-티켓의 Total Amount(KRW)를 선결제(원화) 지출 항목으로 추가
- 텍스트 추출은 `pdf.js`를 PDF 사용 시에만 CDN에서 동적 로드 (최초 1회 인터넷 필요). 스캔본/이미지 PDF는 미지원
- 아시아나 양식으로 검증: 7/25 출국 ~ 7/31 귀국(6박 7일), OZ707/OZ708 시각, 항공료 ₩654,400
- 재인식 시 이전에 PDF로 넣은 비행/항공료 항목만 교체

### 8. 지출 카테고리 / 항목별 / 골프 합산 / 선결제 관리
- 일정 항목마다 **지출 분류**(식비·숙소·교통·마사지·쇼핑·밤일정·골프·기타) 지정 (미지정 시 이름으로 자동 추정)
- 지출 탭을 **카테고리별 / 일차별 / 항목별** 표로 표기, 전체 합계는 **한 줄 바**로 표기
- 골프 라운드를 **일정 일차에 배정**해 타임라인에 자동 표시하고, 1인 비용을 지출 합계에 합산
- **선결제 / 기타 지출 추가·삭제**: 미리 결제한 항공료·숙소·골프 등을 직접 입력. 항목별 **통화(₩/₱/$) 선택** → 환율로 환산해 합산

## 데이터 관리
- **PDF로 저장 / 공유**: 모든 탭을 한 문서로 펼쳐 인쇄
- **백업 내보내기 / 가져오기**: 일정·지출·환율·골프·여행기간·선결제를 JSON으로
- **완전 초기화 (전체 삭제)**: 모든 데이터를 삭제하고 **빈 상태**로 시작 (기본 일정 복원 아님)
- 이번 버전에서 최초 실행 시 기존 저장 데이터를 1회 자동 삭제하도록 적용 (빈 플래너로 시작)
